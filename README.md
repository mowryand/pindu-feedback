# Pindu Feedback

This is the issue tracker for the **Pindu Anki Add-On**, a reading engine and interface for language acquisition built on top of Anki.

- [Get Pindu on AnkiWeb](https://ankiweb.net/shared/info/600135790)
- [Pindu's Website](https://www.readpindu.com)

## Reporting an Issue

Use the [Issues tab](https://github.com/mowryand/pindu-feedback/issues) to:

- **Report a bug** — something isn't working as expected
- **Request a feature** — an idea for improving Pindu

Before opening a new issue, please check if a similar one already exists.
